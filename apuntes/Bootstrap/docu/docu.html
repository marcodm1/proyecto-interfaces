12 columns

Además de col para columnas de ancho automático, podemos ajustar diferentes tamaños de columna con el de 12 columnas.

Usa el sistema col-{size} donde {size} es un número de 1 a 12.

Responsive grid system

Bootstrap usa flexbox internamente y es responsivo (se adapta a las dimensiones del vispositivo/viewport).

Breakpoints: se definen 5 breakpoints con unos tamaños de píxeles con nombres. Los tienes en la tabla de Grid Options: extra small, small, medium, large y extra large.

Las columnas con col o col-{size} funcionan igual para cualquier tamaño de viewport.

Las columnas con col-{breakpoint} o col-{breakpoint}-{size} van a ocupar:

  • su tamaño correspondiente (automático o {size}) si el viewport es menor que el breakpoint.
  • una fila completa si el viewport es mayor o igual al breakpoint.

Margin y padding

Desde v4 hay clases para facilitarlo (ver documentación):

  • m-3 añade un margin de tamaño 3 (según lo que Bootstrap defina como tamaño 1)
  • p-1 padding de tamaño 1
  • ml-3 margin left. Similar para right, top y bottom: mr-3, mt-3 y mb-3
  • mx-3 margen eje X (left, right)
  • my-3 margen eje Y (top, bottom)
  • mx-auto margen left y right auto (centrado clásico)

Y todas las combinaciones que se te ocurran...

Todos los márgenes y paddings son responsive, así que puedes hacer:

  • mx-md-5 margen eje X de 5 para viewports medium o mayor.
  • pl-sm-1 pl-md-3 paddings diferentes para viewports small y medium

Todos estas clases de margin y padding los puedes usar dentro de columnas para dejar espacios dentro (en este contexto el espaciado se llama gutter, pero es lo mismo).

En un grid también puedes quitar todos los paddings con no-gutter.
